conditional statements are used to perform different actions based on different conditions.
if statement - use this statement to execute some code only if a specified condition is true;

syntex 
if (condition) {
    code to executed if condition is true 
}

if ... else statement - use this statement to execute some code if the condition is true

and another code if the condition is false ;

syntex 
if (condition) {
    code to executed if condition is true
}

else {
    code to executed if the condition is false or not true 
}
